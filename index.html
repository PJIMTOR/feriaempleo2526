Las 4 Actividades del Taller

Hackea tu web favorita

Usamos el inspector del navegador para modificar HTML y CSS en tiempo real. Cambiamos títulos, colores y textos para entender cómo el servidor envía el código al navegador.

Detectar tecnologías

Analizamos qué tecnologías usan las webs reales (React, WordPress, Nginx…). Descubrimos cómo identificar frameworks y servidores en segundos.

Peticiones HTTP en directo

Visualizamos cómo el navegador envía peticiones GET y recibe respuestas 200 o 404. Entendemos cómo funciona realmente Internet.

Publica tu CV en Internet

Generamos un CV en HTML con IA y lo desplegamos en un servidor gratuito. En pocos minutos cada alumno tiene su propia web pública online.
