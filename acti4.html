Publica tu CV en Internet

Aprende a crear tu primer CV online con IA y subirlo a la web en minutos.

🎯 Objetivo

Generar un CV en HTML, guardarlo y publicarlo para que cualquiera pueda verlo online.

Paso 1 Generar el CV con IA

Usa un asistente de IA (como ChatGPT) y pide algo así:

Créame un CV profesional en HTML moderno con:
- Nombre: Juan Pérez
- Estudios: ESO
- Intereses: Tecnología y videojuegos
- Habilidades: Trabajo en equipo, aprendizaje rápido
- Diseño moderno con CSS integrado

La IA generará un archivo HTML listo para usar.

Paso 2 Guardar como .html

  1. Copia todo el código HTML que te da la IA.
  2. Abre un editor de texto (Bloc de notas, VSCode, Sublime…)
  3. Pega el código y guarda el archivo como nombre-apellido.html (sin espacios ni tildes).
  4. Verifica que al abrirlo en el navegador se vea correctamente.

Paso 3 Desplegar en internet con GitHub Pages

Vamos a publicar tu CV dentro del repositorio del proyecto para que aparezca automáticamente en la web.

  1. Entra en el repositorio de GitHub feriaempleo2526.
  2. Accede a la carpeta /cv.
  3. Sube tu archivo HTML y confirma con Commit changes.

Una vez subido, tu CV aparecerá automáticamente en la página de CV publicados. Podrás acceder a él en:

https://pjimtor.github.io/feriaempleo2526/cv/nombre-apellido.html

Paso 4 Tu CV está online

¡Listo! Ahora puedes compartir tu CV con amigos, profesores o en redes. Cualquiera con el enlace podrá verlo desde cualquier dispositivo.

Es tu primer proyecto web publicado en Internet.
