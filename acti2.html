Actividad 2 🔍
Detecta qué tecnología usa una web

Descubre qué hay detrás de las páginas que visitas cada día.

🎯 Objetivo

Aprender a identificar qué tecnologías utiliza una web real: frameworks, CMS y servidores.

🛠 Opción 1: Usar herramientas

Instala una extensión como Wappalyzer y entra en cualquier web. Verás si usa React, WordPress, Angular, Nginx, etc.

🧠 Opción 2: Ver código fuente

Clic derecho → Ver código fuente y busca pistas como:

  • /wp-content/ → WordPress
  • react.production.min.js → React
  • ng- → Angular

💡 ¿Qué estás aprendiendo?

Que cada web tiene una arquitectura tecnológica detrás. En DAW aprenderás a construir exactamente eso.
