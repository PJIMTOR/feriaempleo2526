Actividad 1 🚀
Hackea tu Web Favorita

Descubre cómo funcionan realmente las páginas web usando el Inspector del navegador.

🎯 Objetivo

Entender que las páginas web están hechas de HTML y CSS, y que el navegador solo muestra lo que el servidor le envía.

🛠 Paso 1: Abre el Inspector

Entra en cualquier web (por ejemplo Google o Amazon), haz clic derecho y selecciona “Inspeccionar”.

✏ Paso 2: Modifica el HTML

Busca un título dentro del código y haz doble clic para cambiarlo.

<h1>Amazon</h1> ↓ Cambiar por ↓ <h1>Mi Tienda Online</h1>

¡El cambio aparece al instante!

🎨 Paso 3: Cambia el CSS

Desde el panel de estilos puedes modificar colores, tamaños y márgenes.

color: red; font-size: 40px; background-color: yellow;

💡 ¿Qué está pasando realmente?

No estás hackeando la web real. Solo estás modificando la copia del HTML que tu navegador recibió del servidor.

Si recargas la página… todo vuelve a la normalidad.

🧠 Lo que estás aprendiendo

  • ✔ Cómo está estructurada una página web
  • ✔ Cómo funciona el frontend
  • ✔ Qué hacen realmente los desarrolladores web
